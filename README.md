O **NutriAI** é um aplicativo web focado em dieta, registro de calorias, monitoramento de macronutrientes e rastreamento de atividades físicas. Focado em uma experiência premium nativa *(Mobile-First)*, o aplicativo oferece um design moderno, animações ricas e interações imersivas diretamente pelo navegador.

---

## 🌟 Recursos Principais

- **Diário de Alimentação Inteligente:** Registre seus alimentos diários explorando um banco de dados integrado, monitorando proteínas, carboidratos e gorduras com base nos seus objetivos com controles dinâmicos "Ring Progress".
- **Gerador de Treinos assistido por IA:** Monte a rotina perfeita definindo dias da semana, equipamentos e limitações. Tudo projetado com uma interface intuitiva de múltipla escolha.
- **Scanner Inteligente via Câmera:** Identifique as calorias e macronutrientes dos seus alimentos utilizando a câmera do próprio celular — integração direta na web!
- **Painel de Controle Elegante (Dashboard):** Acompanhe tendências semanais, balanço hídrico diário e a evolução de suas metas calóricas com gráficos interativos de arco.
- **Gestão de Perfil & Preferências:** Controle sobre seu estilo visual (Tema Claro, Tema Escuro), ajustes de conta, troca dinâmica do idioma do app e acompanhamento do peso alvo.

## 🛠️ Tecnologias Utilizadas

O desenvolvimento foi feito sem a dependência pesada de frameworks em prol de performance e leveza extrema, garantindo que o aplicativo carregue instantaneamente.
- **HTML5** Estrutura semântica com layout de *Single Page Application*.
- **CSS3** Tipografia responsiva, variáveis de ambiente, estilizações animadas (`@keyframes`, propriedades *glassmorphism* e sombras customizadas).
- **JavaScript Puro (Vanilla JS)** Gestão do roteamento das telas (`switchTab`), APIs Nativas (WebRTC para câmera) e manipulação dinâmica de DOM.
- **LocalStorage:** Todo o progresso do usuário durante o dia é armazenado localmente para garantir o máximo de privacidade e o uso off-line do banco de dados recente.

## 🚀 Como Executar

### Pré-requisitos
Por ser estruturado puramente na tríade essencial da web (HTML/CSS/JS), você não precisará rodar longas instalações de dependências (NPM).

1. Clone o repositório em sua máquina:
   ```bash
   git clone https://github.com/SeuUsuario/NutriAI.git
   ```

2. Abra acesso root do projeto.
3. Para ativar a câmera do aplicativo web em ambiente local no celular e evitar o bloqueio de segurança dos navegadores para uso de *mediaDevices*, utilize a extensão **Live Server** (do VS Code) ou similar para rodar por uma conexão *localhost*.
